# Introduction

Anyone can learn to solve the Rubik's Cube!
In this guide, you'll learn how to solve using the Roux Method.
No experience with the Rubik's Cube is needed.
We'll cover the basics and learn a beginner-friendly version of Roux.
Let's get started!
